# Cortex-Path

AI-powered codebase understanding and visualization platform built during the CodeKada 2026 Hackathon.

## The Problem

Developers waste hours trying to understand unfamiliar codebases.

Whether onboarding to a new team, revisiting an old project, or reviewing client work, navigating hundreds of files and folders can significantly slow development.

Cortex-Path was built to solve that problem by helping developers understand an entire codebase in seconds.

---

## Features

* Analyze project structure automatically
* Understand file and module relationships
* Generate an interactive codebase visualization map
* Surface architecture insights
* Accelerate developer onboarding
* Explore large repositories with clarity

## How It Works

1. Upload a local codebase
2. Cortex-Path analyzes the repository structure
3. AI processes dependencies and relationships between files
4. An interactive visualization map is generated for exploration
